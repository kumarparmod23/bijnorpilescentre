# Bijnor Piles Centre — Auto-Blog Repository

A fully automated bilingual (English + Hindi) medical blog for **Bijnor Piles Centre**, hosted **free** on GitHub Pages. Two SEO-optimized posts publish every day at 06:00 IST and are automatically submitted to Google & Bing for instant indexing.

---

## 1. Quick Start

```bash
# 1. Create a new GitHub repo named: bijnorpilescentre
# 2. Upload all the files in this folder to that repo (preserve structure)
# 3. Add the secrets listed below
# 4. Settings → Pages → Source: GitHub Actions
# 5. Actions tab → "Daily Auto Blog" → Run workflow (test)
```

## 2. Required GitHub Secrets

Settings → Secrets and variables → Actions → New repository secret

| Secret | Where to get it |
|---|---|
| `ANTHROPIC_API_KEY` | https://console.anthropic.com → API Keys |
| `GOOGLE_INDEXING_JSON` | Google Cloud Console → IAM → Service Accounts → JSON key (entire file contents). Enable **Indexing API**. |
| `INDEXNOW_KEY` | https://www.bing.com/indexnow (generate any 32-char hex string and host it at `/INDEXNOW_KEY.txt`) |

## 3. Google Search Console setup

1. Add property `bijnorpilescentre.com` (DNS verification).
2. Submit sitemap: `https://bijnorpilescentre.com/sitemap.xml`
3. Add the **service account email** (from the JSON key) as **Owner** in Search Console settings → Users and permissions.

## 6. How it Runs

1. **00:30 UTC daily** GitHub Actions triggers `daily-blog.yml`.
2. `generate_blogs.py` reads today's topic from `_data/calendar.json`, calls Claude API and writes one English + one Hindi `.md` file with full SEO front-matter.
3. The workflow commits the new posts. The push triggers `jekyll.yml` which rebuilds the site and deploys to GitHub Pages.
4. After a 90-second wait, `submit_to_google.py` pings Google Indexing API + Bing IndexNow + sitemap pings.

## 7. Customise Before You Go Live

- Edit `_config.yml` → replace `[Your Name]`, phone, WhatsApp.
- Edit `_includes/cta.html` → set real phone numbers.
- Edit `about.md` and `contact.md` with your full details.
- Replace `assets/images/clinic.jpg` with a real clinic photo.
- (Optional) Extend `_data/calendar.json` from 30 to 365 unique topics.
